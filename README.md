# CAB・GAB対策ラボ

CAB・GAB対策ラボは、就職活動で実施される適性検査 **CAB / GAB** の基礎練習を、
ブラウザ上で手軽に行うためのシンプルな学習サイトです。

## このサイトは何をするサイトか

このサイトでは、次のような対策を短時間で行えます。

- **CAB対策（四則逆算）**
  - 方程式や逆算系の問題を解き、計算のスピードと正確性を高めます。
- **CAB対策（法則性）**
  - 数列・文字列の規則を見抜く問題で、パターン認識力を鍛えます。
- **GAB対策（言語理解）**
  - 語彙や意味関係の問題を通して、言語的な判断力を確認します。

また、ミニ演習では次の機能を使えます。

- 分野を選んで問題を開始（すべて / 四則逆算 / 法則性 / 言語理解）
- 選択肢をクリックして即時に正誤フィードバックを確認
- 問題のスキップ
- 演習終了後に、正答率と苦手分野（ミス回数）を表示

## サイトへのアクセス方法（ローカル起動）

> 以前の `/workspace/CABGABtest` はこの実行環境内のパスで、
> **Windowsローカル環境ではそのまま使えません**。
> 必ず「自分のPC上でクローンした場所」に移動してください。

### 1. リポジトリをクローン（未取得の場合）

```bash
git clone <このリポジトリのURL>
```

### 2. クローンしたフォルダへ移動

#### Windows（コマンドプロンプト例）

```bat
cd %USERPROFILE%\CABGABtest
```

#### macOS / Linux（ターミナル例）

```bash
cd ~/CABGABtest
```

> フォルダ名を変更した場合は、その名前に読み替えてください。

### 3. ローカルサーバーを起動

Python が使える環境で、次のコマンドを実行します。

```bash
python -m http.server 4173
```

> ポート番号 `4173` は例です。必要に応じて変更できます。

### 4. ブラウザでアクセス

以下のURLをブラウザで開きます。

- http://localhost:4173

トップページが表示されたら、`問題を開始` ボタンから演習できます。

## ファイル構成

- `index.html` : ページ構造（ヒーロー、説明、クイズ、結果表示）
- `styles.css` : レイアウトと見た目
- `script.js` : 問題データ、出題制御、正誤判定、結果集計
